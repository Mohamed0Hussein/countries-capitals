# Country Capital Matching Game

## Overview

This is a simple web-based game where you can match countries with their corresponding capitals. The goal is to match as many pairs as possible within a given time limit.

## Features

- Randomized country-capital pairs for each game.
- Timer to track the time spent on the game.
- Click on a country or capital to select it, then click on its matching pair to make a match.
- The game ends when all pairs are matched, displaying a "Congratulations, you WON" message.
- You can try the game again by clicking the "Try again" button.

## Technologies Used

- React: This game is built using React, a popular JavaScript library for building user interfaces.
- React Router: Used for navigation within the game.
- HTML and CSS: For the structure and styling of the game.

## Getting Started

To run this game on your local machine, follow these steps:

1. Clone this repository:

   ```bash
   git clone https://github.com/your-username/country-capital-game.git
   ```

2. Change to the project directory:

   ```bash
   cd country-capital-game
   ```

3. Install the required dependencies:

   ```bash
   npm install
   ```

4. Start the development server:

   ```bash
   npm start
   ```

5. Open your web browser and access the game at `http://localhost:3000`.

## How to Play

1. Click on a country or capital to select it (the selected item will be highlighted).
2. Click on the matching country or capital to make a pair.
3. If the pair is correct, they will be deleted, and you can proceed to the next pair.
4. If the pair is incorrect, they will be highlighted in red briefly, and you can try again.
5. The game ends when you successfully match all the country-capital pairs.
